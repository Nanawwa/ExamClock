# 倒计时显示优化方案

## 目标
区分三个阶段（考前/考中/考后），大字显示完整描述句，最后 5 分钟读秒，小字不重复大字内容。

## 三个阶段的显示内容

| 阶段 | 大字 | 小字（remain-label） |
|------|------|---------------------|
| 考前 | 距离考试开始还有 **45** 分钟 | 将于 14:00 开始 |
| 考中（>5min） | 距离考试结束还有 **45** 分钟 | 已进行 30 分钟 |
| 考中（≤5min） | 距离考试结束还有 **4:59** | ← 读秒（与大字同步） |
| 考后 | 考试已结束 **12** 分钟 | 于 15:30 结束 |

> 最后 5 分钟时，大字的数字从纯分钟切换为 `分:秒` 格式，同时小字也同步显示秒数，形成双重视觉强调。

## 修改文件
`index.html` — 单文件项目，所有改动集中在此。

## HTML 变更（~L399-403）

将大字区域从单一 `span` 拆分为三段式：

```html
<div class="cell cell-remain">
  <div class="remain-digits">
    <span class="remain-prefix" id="remain-prefix"></span>
    <span class="remain-number" id="remain-value">0</span>
    <span class="remain-suffix" id="remain-suffix"></span>
  </div>
  <div class="remain-label" id="remain-label"></div>
</div>
```

删除旧的 `remain-minutes` span。

## CSS 变更（~L232-247）

替换 `.remain-digits` 和相关样式：

```css
.remain-digits {
  display: flex;
  align-items: baseline;
  justify-content: center;
  gap: 0.12em;
  line-height: 1;
}
.remain-prefix, .remain-suffix {
  font-size: clamp(1.1rem, 3vw, 1.8rem);
  font-weight: 500;
  color: rgba(255, 255, 255, 0.65);
  letter-spacing: 0.04em;
}
.remain-number {
  font-size: clamp(4.5rem, 15vw, 9rem);
  font-weight: 800;
  color: #fff;
  font-variant-numeric: tabular-nums;
  font-family: "Helvetica Neue", "Arial", "DIN Alternate", system-ui, sans-serif;

```

删除旧的 `.remain-digits` 中的 font-size/font-family 等属性（合并到上面）。

小屏 `@media (min-width: 1200px)` 中的 `.remain-number { font-size: 10rem; }` 替换原来的 `.remain-digits`。

## JS 变更（~L503-556）

### computeState 返回值扩展

增加字段：`prefix`、`suffix`、`remainValue`（显示文本）、`remainSec`（秒数）。

```
状态判断：
- now < start → prefix="距离考试开始还有", suffix="分钟", remainValue=分钟数
- now < end, remain>5min → prefix="距离考试结束还有", suffix="分钟", remainValue=分钟数
- now < end, remain≤5min → prefix="距离考试结束还有", suffix="", remainValue="M:SS"
- now >= end → prefix="考试已结束", suffix="分钟", remainValue=已过分钟数
```

### tick 函数更新

```
- remain-prefix.textContent = s.prefix
- remain-number.textContent = s.remainValue
- remain-suffix.textContent = s.suffix
- remain-label:
  - 考前: "将于 HH:MM 开始"
  - 考中 >5min: "已进行 X 分钟"
  - 考中 ≤5min: "M:SS 结束"（与大字同步读秒）
  - 考后: "于 HH:MM 结束"
```
